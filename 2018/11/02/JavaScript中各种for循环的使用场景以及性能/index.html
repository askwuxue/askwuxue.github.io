注意：请注意上述1.1，使用Array()生成数组后又将数组的内容进行了重新赋值。因为Array（1000000）生成的是具有1000000空位的数组。对于数组内的空位，循环的处理不同。ForEach会跳过空位，for of依旧会遍历空位。因此，如果数组内全部都是空位，会发现ForEach速度最快。具体可以参考阮一峰老师的ES6教程
